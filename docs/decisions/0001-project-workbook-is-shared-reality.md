# 0001 项目工作手册是共享现实

- 状态：accepted
- 日期：2026-07-18

## 问题

移动端实现、跨仓库契约、PR 和持久化语义分散时，新会话容易用局部代码或旧材料代替维护者意图。

## 决定

每个会话从 `AGENTS.md → docs/INDEX.md → docs/WORKFLOW.md` 进入。长期语义、决策、当前工作、设计事实和实现证据各有唯一落点；冲突时停止核对。

## 理由

共享入口让核对比猜测便宜，并允许 Agent 只按任务读取必要上下文。

## 后果

文档不是完成记录。长期变化要同步需求或决策，完成事项从 `NOW.md` 删除，代码事实仍由代码和 Git 证明。

## 验证

静态检查所有索引路径存在，并能按固定入口找到产品需求、状态地图、任务合同和 Review 合同。
